# ⚽ Indoor Football Game Management System

A web-based application developed as a Final Year Project for managing indoor football ground bookings. The system allows users to register, book available time slots, and enables administrators to manage grounds, bookings, and users efficiently.

---

## Features

### User
- User Registration & Login
- Browse Available Grounds
- Book Available Time Slots
- View Booking History
- Update Profile

### Admin
- Secure Admin Login
- Manage Grounds (Add, Edit, Delete)
- Manage Bookings
- Manage Users
- Dashboard with Booking Information

##  Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/Indoor-Football-Game-Management-System.git
```

2. Move the project folder to

```
xampp/htdocs/
```

3. Start Apache and MySQL from XAMPP.

4. Import the SQL database into phpMyAdmin.

5. Open your browser and visit

```
http://localhost/project-folder/
```
